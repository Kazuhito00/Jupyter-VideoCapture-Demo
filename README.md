# Jupyter-VideoCapture-Demo
Jupyter notebook上でリアルタイムにカメラ画像を表示するサンプルです。

# Requirement
 
* OpenCV 3.4.2(or later)

# Usage
Jupyter上でJupyter-VideoCapture-Demo.ipynbを開いて実行してください。
